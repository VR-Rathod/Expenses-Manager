Welcome to the Daily Expenses Sharing Application

Objective:

This tutorial will guide you through designing a daily-expenses sharing application. The application allows users to manage and share expenses efficiently, offering multiple splitting methods and generating downloadable balance sheets.

Features:

User Management:

  • Email: Each user must have a unique email address.
  • Name: Users must provide their full name.
  • Mobile Number: Users need to input a valid mobile number.

Expense Management:

Users can manage expenses with the following split methods:

  • Equal: Split expenses equally among all participants.
  • Exact: Specify the exact amount each participant owes.
  • Percentage: Specify the percentage each participant owes, ensuring the total adds up to 100%.

Expense Calculation Examples:

Equal Split: If you go out with 3 friends and the total bill is 3000, each friend owes 1000.

Exact Split: If you go shopping with 2 friends and pay 4299, Friend 1 owes 799, Friend 2 owes 2000, and you owe 1500.

Percentage Split: If you attend a party with 2 friends and one of your cousins, and you owe 50%, Friend 1 owes 25%, and Friend 2 owes 25%.

Balance Sheet:

  • Individual Expenses: Display each user's expenses separately.
  • Overall Expenses: Show the cumulative expenses for all users.
  • Download Feature: Provide an option to download the balance sheet in a convenient format.
